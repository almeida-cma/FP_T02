📊 Fluxograma: Média do Aluno

1. INÍCIO
2. Ler NOTA1
3. Ler NOTA2
4. MÉDIA = (N1+N2)/2
5. MÉDIA >= 7?
SIM → APROVADO
NÃO → REPROVADO
6. FIM
📜 Código (Portugol Webstudio): programa { funcao inicio() { real n1, n2, media escreva("Nota 1: ") leia(n1) escreva("Nota 2: ") leia(n2) media = (n1 + n2) / 2 se (media >= 7) { escreva("APROVADO!") } senao { escreva("REPROVADO!") } } }
